# CorpFlow 智能体团队

一个 CEO + Manager + Worker 三层协作的 AI 团队。

## 智能体结构

```
CEO (战略决策)
  └── Manager (任务管理)
        └── Worker (执行落地)
```

## 使用方式

在对话中直接召唤：
- `@CorpFlow CEO` - 战略决策
- `@CorpFlow Manager` - 任务管理  
- `@CorpFlow Worker` - 执行任务

## 功能

- 📊 OKR 管理 - 目标设定与追踪
- 📈 绩效评估 - AI 自动评估
- 🗳️ 决策投票 - 群体决策
- 📋 任务指派 - 向下派发任务

## 对话示例

```
用户: 帮我制定本季度OKR

CEO: 分析业务目标，拆解成 O 和 KR
  ↓
Manager: 分配给具体负责人
  ↓  
Worker: 执行并定期汇报进度
```

## 权限

- CEO: 全部权限，可查看所有数据
- Manager: 查看本团队数据
- Worker: 查看个人任务
